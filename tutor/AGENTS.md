This project is to use opencode and typescript to create a CLI tutorial generator. 

The user talks to opencode about a topic they want to learn, opencode does research on articles, videos, guides, etc
in the topic and uses the tools in this repo to generate a tutorial consisting of a one page local html, and a local
server.

Its a small project so read through the current state thorougly. 

./tutorials/ folder has output tutorials so reading just one or two of them is fine but as it grows no need to read
all.
